# Velora

Velora is a conversational AI framework for Go. It provides a simple and extensible way to build chatbots and other conversational applications.

## Features

* **Conversational AI:** Build chatbots and other conversational applications.
* **Extensible:** Easily add new features and integrations.
* **Simple:** Get started quickly with a simple and intuitive API.

## Getting Started

To get started with Velora, you'll need to have Go installed on your machine. You can then install the Velora CLI by running the following command:

```
go get github.com/velora-chat/velora/cmd/velora
```

Once you have the Velora CLI installed, you can create a new project by running the following command:

```
velora new my-project
```

This will create a new directory called `my-project` with a basic Velora project structure. You can then run the project by running the following command:

```
cd my-project
velora run
```

This will start the Velora development server. You can then open your browser to `http://localhost:8080` to see your chatbot in action.
